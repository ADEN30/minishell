# Cub3D

## Description

Projet `Cub3D` de l'école 42. L'objectif est de créer un moteur graphique simple en utilisant la bibliothèque `MinilibX` et d'implémenter un jeu de type Wolfenstein 3D.

## Fonctionnalités supportées

- Affichage d'une carte en 3D.
- Gestion des textures et des couleurs.
- Déplacement dans l'environnement en vue à la première personne.
- Moteur graphique avec lancer de rayons.

## Comment utiliser

1. Clonez le dépôt : `git clone <lien du dépôt>`.
2. Naviguez dans le répertoire `cub3d`.
3. Exécutez `make` pour compiler l'exécutable `cub3D`.
4. Vous pouvez maintenant utiliser l'exécutable `cub3D` pour explorer l'environnement.

## Exemple

```bash
./cub3D maps/map.cub
